Category Theory Summer Course 2019

by David Forsman

Description

Quite exactly one year ago a group of enthusiastic mathematicians started the Category Theory Reading Group which has been active since then. Because reading category theory independently is a challenging task and there is a growing interest in the topic, this year there will be a lectured course about category theory.

Category theory is a field of mathematics in which a language is created to look for very general patterns that permeate through all of mathematics. Category theory gives a new, very algebraic lense to view mathematics. Itâ€™s a lot of fun but it requires that one has studied the object which we put into the categories.

The topics are the same as in Introduction to Category Theory I, but made a bit easier. The outline of the course is following:

  • Definition of category
  • Category theoretical constructions
  • Functors and natural transformations
  • Either limits in category theory or Yoneda's lemma

Prerequisites

It would be necessary to know basic set theory and especially intersections and unions over different (possibly uncountable infinite) indexing sets. The ideas from abstract algebra, such as groups, subgroups, quotient groups, homomorphisms and equivalence relations will come up. It would be nice if people know that a continuous function can be defined as a function such that the inverse image of any open set is an open set.

Either way, everybody who is interested in the topic is welcome to join our study group or just to visit the lectures!
